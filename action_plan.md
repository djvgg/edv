## 1. Bracket Assignment to Tables
- Implement drag-and-drop or selection to assign brackets to specific tables (table 1, 2, 3, 4).
- Visually indicate which bracket is assigned to which table in the UI.

## 2. Bracket Details View
- When a bracket is selected or clicked, show a detailed view above the tables (as in the wireframe).
- Include match details, participant info, and allow editing if needed.

## 3. Search and Filter Improvements
- Enhance the search bar to filter by table assignment, gender, or other criteria.
- Optionally, add sorting or advanced filtering.

## 4. Manual Override and Validation
- Add an “Override valid” button to manually mark brackets as valid/invalid.
- Show validation status in the UI.

## 5. System Controls
- Implement the “Close the system” button to safely exit or lock the interface.

## 6. Data Export/Import
- Add options to export bracket assignments and results (e.g., to Excel or PDF).
- Allow importing updated participant lists or results.

## 7. Polish and Usability
- Improve UI layout and responsiveness.
- Add tooltips, help, and error messages for better user experience.

Start with bracket-to-table assignment, then proceed to bracket details view and validation features.
